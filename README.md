## setup

```bash
npm install
npm run dev

```

in another terminal

if you first run this project, please run it at first.
```bash
npm run prepack
```

then run

```bash
npm start

```

## pack app

```bash
npm run build

```

* in mac 

```bash
npm run packer:mac
```

* in windows

```bash
npm run packer:win
```
